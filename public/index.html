国会MCPサーバ

国会会議録・帝国議会会議録を検索できるMCPサーバー

国立国会図書館が提供する2つのAPIを、AIアシスタントから直接利用できるリモートMCPサーバーです。帝国議会と現行の国会を横断的に調べることもできます。議事録ではすべての発言に固有のURLが与えられているので、それを示してくれと指示すれば、URLを出力します。

データソース対象期間
国会会議録検索システム1947年〜現在
帝国議会会議録検索システム1890年〜1947年

セットアップ

Node.jsのインストールは不要です。URLを指定するだけで利用できます。ClaudeがこのMCPサーバを使おうとしない場合は、「国会MCPを使って調べて」と命じてください。

ブラウザ版Claude

画面下の⚙️設定画面から［コネクタ］を選択。一番下の［カスタムコネクタを追加］をクリック。［リモートMCPサーバURL］という入力欄があるので、「http://kokkai-mcp.vercel.app/mcp」と入力。［名前］は「国会MCP」にしておいて下さい。モバイルアプリやデスクトップアプリでも使えるようになります。

Claude Desktop

claude_desktop_config.jsonに以下を追加してください。

{
  "mcpServers": {
    "kokkai": {
      "type": "url",
      "url": "https://kokkai.vercel.app/mcp"
    }
  }
}

Claude Code

claude mcp add kokkai --transport http https://kokkai.vercel.app/mcp

利用できるツール

ツール説明
search_kokkai_meetings国会の会議録を検索（本文なし、最大100件）
get_kokkai_meeting国会の会議録を取得（全発言の本文付き、最大10件）
search_kokkai_speeches国会の発言を検索（本文付き、最大100件）
search_teikoku_meetings帝国議会の会議録を検索（本文なし、最大100件）
get_teikoku_meeting帝国議会の会議録を取得（全発言の本文付き、最大10件）
search_teikoku_speeches帝国議会の発言を検索（本文付き、最大100件）
search_all_meetings国会と帝国議会を横断して会議録を検索
search_all_speeches国会と帝国議会を横断して発言を検索
count_kokkai国会の検索結果件数を取得
count_teikoku帝国議会の検索結果件数を取得
count_all国会＋帝国議会の合計件数を取得
get_all_kokkai_meetings国会の会議録を全件取得（最大1,000件）
get_all_teikoku_meetings帝国議会の会議録を全件取得（最大1,000件）

使用例

  • 「人工知能」に関する国会での発言件数を調べる → count_kokkai
  • 戦前から現在まで「教育」に関する議論を横断検索 → search_all_speeches
  • 特定の議員の発言を検索 → search_kokkai_speeches
  • 予算委員会の会議録一覧を取得 → search_kokkai_meetings

注意事項

  • 本サーバーが利用するAPIは国立国会図書館が提供しています。短時間での大量アクセスはお控えください
  • 全件取得ツールはリクエスト間に1秒の間隔を設けており、最大1,000件（10ページ）で打ち切られます
  • 発言の著作権は個々の発言者に帰属します
  • 各ツールのレスポンスはJSON形式です
